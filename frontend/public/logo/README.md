# Eudox AI Logo Assets

This folder contains the logo files for Eudox AI.

## Files

- `logo-small.svg` - Small version of the logo (for headers, navbars)
- `logo-large.svg` - Large version of the logo (for landing pages, sign-in)
- `logo-small.png` - PNG version of small logo
- `logo-large.png` - PNG version of large logo

## Usage

### In React Components

```jsx
// Small logo
<img src="/logo/logo-small.svg" alt="Eudox AI Logo" />

// Large logo
<img src="/logo/logo-large.svg" alt="Eudox AI Logo" />
```

### Logo Colors

- Primary: `#5fc3d8` (Turquoise)
- Secondary: `#4db8d1` (Darker Turquoise)
- Accent: `#a8e6cf` (Light Mint)

## Design Guidelines

- Maintain aspect ratio when resizing
- Use on white or light backgrounds for best visibility
- Minimum size: 32px height for readability
